# Flask Budget Tracker with User Authentication
A web app where users can register, log in, and track their own personal expenses. Each user only sees their own data.

## Features
- It allows the adding and removal of expenses 
- It uses a SQLite database to store user and expense data.
- It has a simple and clean user interface, with a focus on functionality and ease of use.

## Tech stack
- Python
- Flask
- SQLite
- HTML/CSS
- JavaScript

## How to run it
Clone the repository
1. Install the required dependencies using pip install -r requirements.txt
2. Run app.py in your terminal
3. Open your web browser and navigate to http://127.0.0.1:5000
4. Create an account or sign in with an existing account
5. Start tracking and adding expenses.
